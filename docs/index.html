jaco

Japanese string and charactor converter.

install

$ npm install jaco

Usage

var jaco = require('jaco');
var Jaco = jaco.Jaco;

jaco.hiraganize('カタカナ'); // => かたかな
jaco.katakanize('かたかな'); // => カタカナ

var jStr01 = new Jaco('かたかな');
jStr01.toKatakana(); // => カタカナ
