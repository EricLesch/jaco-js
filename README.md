jaco
====

Japanese string and charactor converter.

[![Code Climate](https://codeclimate.com/github/YusukeHirao/jaco.png)](https://codeclimate.com/github/YusukeHirao/jaco)

## install

```sh
$ npm install jaco
```

## Usage

```javascript
var jaco = require('jaco');

jaco.katakana('かたかな'); // => カタカナ
```
